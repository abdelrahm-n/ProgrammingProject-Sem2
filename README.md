# Stageplatform

Platform voor stagebeheer bij EhB Toegepaste Informatica.

## Projectstructuur

- `frontend/` - statische HTML, CSS en JavaScript
- `backend/`  - Node.js/Express API
- `database/` - SQL scripts voor de database

## Installatie backend

```bash
cd backend
cp .env.example .env
npm install
npm run dev
```
